Airline Passenger Sentiment Analysis

A Flask-based Machine Learning web application that analyzes airline passenger reviews and predicts whether the sentiment is positive or negative using NLP techniques.

Features:
- Sentiment prediction for airline reviews
- Machine Learning based classification
- NLP with TF-IDF Vectorization
- Simple and clean web interface
- SQLite database integration

Technologies Used:
- Python
- Flask
- Scikit-learn
- NLTK
- SQLite
- HTML/CSS

Example Input:
The airline service was excellent and staff were very friendly.
Prediction:
Positive Sentiment

Future Improvements:
- Multi-class sentiment analysis
- Airline rating dashboard
- Deep Learning models
- Live airline review scraping
